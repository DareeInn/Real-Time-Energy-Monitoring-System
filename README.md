# ⚡ Real-Time Energy Monitoring System

A full-stack IoT-based energy monitoring system that tracks and visualizes real-time electrical metrics using a Raspberry Pi and a Shelly smart plug.

---

## 🚀 Features

- ⚡ Real-time power, voltage, and current monitoring  
- 📊 Live data visualization dashboard  
- 🌡️ Temperature tracking  
- 🔌 IoT integration with Shelly smart plug  
- 🖥️ Raspberry Pi-based local server  
- 📈 Historical data tracking (SQLite)

---

## 🧠 How It Works

Shelly Smart Plug → API → Flask Backend → SQLite → Web Dashboard

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite  
- **Hardware:** Raspberry Pi, Shelly Smart Plug  
- **Communication:** REST API / MQTT  

---
